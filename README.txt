使用方法：
将ddp_client.py解压到某文件夹下，在同一目录下打开命令行cmd窗口，输入命令：python ddp_client.py 2409:8a20:1ca5:6be0:5806:56be:f0c:a29f
打回车，命令行显示'Connected'和服务器发送的问候语即表示连接成功，然后等待游戏开始。

提示：
这是两人模式的ddp游戏。游戏规则我假定你们都了解了。
第一，确保你的电脑上又python3，如果没有，下载一个为好。
其次，确保你的电脑上有ipv6，如果没有，建议用手机开一个热点，用电脑连接上，这样就有ipv6了。（当然，你也可以尝试自己配置，不过小心断网）
一定要确保你每次出牌的时间不得超过20秒（指的是发到我服务器上的时间，当然，我不信你延迟能超过2秒），否则服务器就自动判定你退出了。
若输入如上的命令连接失败，
第一种可能是我没有开服务器，可以在QQ上要求我打开服务器；
第二种可能是上述的ipv6地址过期了，可以在QQ找我要新的ipv6地址；
第三种可能是人已经满了，这种有一个明显的特点，就是只显示'Connected'不显示问候语（服务器现在只能同时容纳两个人，这点需要继续完善）；
第四种可能是你自己的问题，可以QQ私聊我。
第五种可能是你发现了服务器的bug，你是大功臣，也要来QQ私聊我。

总之，有问题QQ找我就行了！
